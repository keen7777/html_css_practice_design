Problem with div:
1, always gaps between elements or due to the space in div-/div,margin, padding; always need to careful,
1-1, ❗example: container=300px, element1=50px, element2=250px, but it will probably overflow and continue in 2nd line

2, ⭐Use CSS grid to create perfectly aligned horizontal layouts
2-1, Grid is a layout that has rows and columns,2 by 3 = 2 rows and 3 columns
2-2, grid is div has style: display grid
2-3, it removes the gaps as we us block/inline block; and if more texts are here automatically aligned
2-4, if we have more divs than defined columns, then automatically start a new row.
2-5, for the column value, could be fix as xxxpx, or take the remaining space, 1fr,
2-6, for multiple frs , like 1fr 1fr 2fr = 0.25 : 0.25: 0.5 of the screen.
3,⭐ how to set upperlimit/lowerlimit of display element number as zoom in/out?
3-1, notice that outter grid container means all the elements(rows) together have a height,
3-2, but a height in a grid it self means only this gird take that much height.
3-3, make sure check both inner and outter container and if necessary, nested different scale of grids.
